# Quant Research Project

## Overview
This project implements a portfolio optimization and backtesting system using Python.

## Features
- Multi-asset data ingestion (yfinance)
- Return and volatility modeling
- Mean-variance optimization (SciPy)
- Walk-forward backtesting
- Portfolio performance evaluation

## Tech Stack
Python, pandas, numpy, scipy, matplotlib

## Key Results
- Efficient frontier estimation
- Sharpe ratio optimization
- Backtested portfolio performance

## How to Run
pip install -r requirements.txt
